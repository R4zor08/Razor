# Razor AI

Modular voice assistant with local AI integration.

## Project Structure

```
razor-ai/
├── main.py              # Entry point
├── config.py            # Configuration
├── core/                # Assistant orchestration
├── voice/               # Speech I/O
├── system/              # OS automation
├── ai/                  # LLM integration
├── assets/              # Audio & logs
└── utils/               # Shared utilities
```

## Getting Started

```bash
python main.py
```
